# 🍰 Sistema Doceria Mobile

Aplicativo mobile desenvolvido para gerenciamento de uma doceria, focado em agilidade no atendimento, controle de vendas e organização operacional.

O projeto passou por uma reestruturação completa, migrando de uma versão inicial em Python para uma implementação nativa Android utilizando Kotlin.

---

# 📱 Tecnologias Utilizadas

* Kotlin
* Android Studio
* Jetpack Compose
* Material Design
* Gradle

---

# 🚀 Evolução do Projeto

## Primeira versão

A primeira implementação foi desenvolvida utilizando Python, com foco em testes de interface e estruturação inicial do sistema.

## Segunda versão (Atual)

O sistema foi recriado utilizando Kotlin para Android nativo, permitindo:

* Melhor desempenho
* Interface mais moderna
* Maior escalabilidade
* Estrutura preparada para API
* Melhor experiência mobile

---

# 🛠 Objetivo do Projeto

O objetivo é criar um sistema mobile simples, moderno e escalável para pequenas docerias e comércios locais, permitindo gerenciamento rápido diretamente pelo celular.
